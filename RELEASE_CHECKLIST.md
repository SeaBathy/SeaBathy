# SeaBathy release checklist

## Before GitHub publish

- [ ] Confirm app builds with `./gradlew assembleDebug`
- [ ] Test MBTiles loading
- [ ] Test basemap switching
- [ ] Test layer opacity
- [ ] Test waypoint create/edit/delete/search/jump
- [ ] Test measurement tool
- [ ] Test GPS off/locate/follow modes
- [ ] Test depth display modes
- [ ] Confirm no MBTiles files are committed
- [ ] Confirm no APK/build files are committed
- [ ] Add screenshots
- [ ] Create GitHub repository
- [ ] Push main branch
- [ ] Create first release
